# 03 · Spacing System — RenCarApp

Uygulama genelinde kullanılacak boşluk ve boyut standartlarını tanımlar.

---

## Spacing Scale

| Token | Değer |
|--------|-------|
| XS | 4dp |
| S | 8dp |
| M | 12dp |
| L | 16dp |
| XL | 24dp |
| XXL | 32dp |

---

## Corner Radius

| Eleman | Radius |
|---------|---------|
| Chip | 8dp |
| Card | 16dp |
| Bottom Sheet | 24dp |
| Button | 16dp |

---

## Elevation

| Eleman | Değer |
|---------|--------|
| Card | 2dp |
| Dialog | 6dp |
| Bottom Sheet | 8dp |

---

## Kurallar

- Rastgele padding kullanılmamalıdır.
- Boşluklar spacing tokenları üzerinden verilmelidir.
- Radius değerleri ortak tasarım sistemine uygun olmalıdır.
